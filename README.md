2. 본인 branch를 생성해 주세요



```bash
git branch 본인이 원하는 브랜치 이름
git checkout 브랜치 이름

ex)
git branch sw
git checkout sw

(작업 후)

git add .
git commit -m '~~~'

git push origin sw

git check out main (메인으로 빠져나오기)
```

<br>

3. 이제부터 작업은 본인의 브랜치에 커밋!
